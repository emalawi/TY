Lesson 1: HTML Intro

HTML is the structure of a webpage. Use <html> and <body> tags.

<html> <body>Hello World</body> </html>

Lesson 2: CSS Basics

CSS styles your webpage. Use selectors and properties.

<style> body {background-color:lightblue;} </style>

Lesson 3: JavaScript Basics

JavaScript adds interactivity. Use <script> tag and functions.

<script> alert('Hello World!'); </script>
